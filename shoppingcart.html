Smart Shopping Cart

Reduced checkout waiting time using automation.

Project Overview

This project presents a smart shopping cart system that integrates Raspberry Pi and RFID technology to improve the shopping experience and reduce checkout time. RFID tags and readers automatically detect items placed in the cart, updating a virtual inventory and enabling real-time billing. The system helps retailers monitor stock levels efficiently while minimizing out-of-stock situations. It addresses long billing queues commonly seen in busy shopping malls, especially during weekends and discount seasons. An LCD display shows item details and total cost, while Wi-Fi communication allows billing directly from the trolley, saving time for customers.

Design Methodology

The smart shopping cart was designed in Autodesk Fusion 360 with an ergonomic and functional structure integrating RFID, weight sensors, and a touchscreen interface. The model includes optimized storage compartments and modular attachments to improve usability and item safety. Fusion 360 enabled precise modelling and collaborative improvements, resulting in a practical and user-centric prototype.

Block Diagram

The smart trolley system is powered by a Raspberry Pi connected via Ethernet or Wi-Fi for network communication. It uses a load cell weight sensor, camera-based barcode scanner, toggle switches for add/remove/billing operations, and a 16×2 LCD to display product details. The system operates on 3.3V and 5V power supply, enabling real-time item detection and billing functionality.
